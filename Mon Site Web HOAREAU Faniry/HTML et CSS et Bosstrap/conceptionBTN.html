Conception du jeu

Interface

L'interface de ce jeu est faite à partir de la bibliothèque tkinter.

Menu

Le Menu contient:

  • Un Bouton Continuer pour pouvoir continuer la partie commencer
  • Un Bouton Jouer contre Ordinateur qui commence une nouvelle partie
  • Un Bouton Jouer en ligne qui permetra bien sûr de jouer en ligne mais n'est pas encore fonctionnel
  • Un Bouton Paramètre qui permet de faire des réglages

Menu

Paramètre contient:

  • Un Slide pour la modification du Volume, le volume qui nécessite la bibliothèques pygame
  • Un Scale pour la modification de la Résolution
  • Un Futur Scale pour la modification de la langue mais elle n'est pas encore fonctionnelle

Structure

Le code source du jeu suit l'architecture MVC(Model View Controler),
le code est donc réparti en trois fichier:

Le fichier controleur: controleur.py

Ce fichier permet de lancer le jeu et est lié au deux autre fichier. Ce fichier permet aussi l'interaction entre l'utilisateur et le jeu, c'est-à-dire que les évenement de lasouris ou du clavier par exemple.

Le fichier vue: view.py

Ce fichier permet d'afficher le jeu (grille, bateaux, explosions, le son, les Boutons,...) dans une fenêtre. La librairie Tkinter y est intégré grace à Python

Le fichier vue: data.py

Ce fichier permet de gérer l'état du jeu (bateau coulé, coordonnées des bateaux, carte de coté 10x10,...) et elle permet aussi de modifier le jeu en générale. La grille est par exemple représenté par une liste de liste de coté 10x10.
